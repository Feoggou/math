Comanda Redimensionare (Meniul Sistem)

Foloseşte această comandă pentru a afişa patru săgeţi ce formează un semn + prin care redimensionezi fereastra activă utilizând tastele săgeţi.

După ce cursorul se schimbă în patru săgeţi ce formează un semn + :

1. Apasă una din tastele săgeţi (stânga, dreapta, sus, sau jos) pentru a muta cursorul către marginea pe care doreşi să o mişti.

2. Apasă una din tastele săgeţi pentru a mişca marginea.

3. Apasă tasta ENTER când fereastra este de mărimea pe care o doreşti.

Note: This command is unavailable if you maximize the window.
