Caseta de dialog Fontul

De aici se aleg opţiunile de formatare a textului documentului.

Lista specificată de Fontul reprezintă lista de familii de fonturi disponibile.

Lista specificată de Mărimea reprezintă mărimea textului de afişat.

În secţiunea Exemplar va fi afişat rezultatul opţiunilor alese, pe un text predefinit. Dacă rezultatul este convenabil, se apasă butonul Acceptă. Dacă se doreşte renunţarea la alegerea unui alt font se apasă butonul Anulează.
